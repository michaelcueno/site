KIT (Matthew Pfister)

This is my design for the kitchen application. The main idea is to let the users feel at home when they use the application. The login screen represents blinds to a window and inbetween each blade is a users name and photo (of their choice). When the user clicks on their login, the blinds will be pulled up and the main pane will be displayed. This pane represents the window behind the blinds, and at first just shows the current weather conditions outside and the temperature. If the user swips to either side (depending upon that users setup) the pane will slide to a different section of the kitchen (i.e. a different pane such as the fridge pane). Each pane has the users custom widgets and each widget has a lesser view (LV) which shows basic functions that, that widget can do. However, if the user wants more detailed functions they can tap and hold the widget and its detailed view (DV) will load in the Main Pane (the window that is displayed after the user selects a login). Panes can be customized with different widgets through the widget manager. When the user goes idle, a screen saver will be loaded (which will cycle by default through family photos).
