1. Notes

In this snap shot of the Notes section, each note is draggable. Each note has a title, and the note text. Each so, each note has an underlined title on the top, followed by the note itself.

Press this "add" button in order add a new note to the notes pane.

This is the add note pane, which appears when the "add" button is pressed. On the left here, we have a list of note templates (only two currently), from which any can be selected. On the right, you can enter the note title and the note text.

Here, pressing the button on the top cancels the whole new note making. To create the note, press the button on the bottom.

2. Calendar

The calendar is a regular calendar. What the different colors mean is explained below:

  • Red box: Sundays
  • Yellow box: Todays and/or Events
  • Blue box: Events
  • Grey box: Days from adjacent months
  • White box: Regular normal days
  • Black Border: Currently selected day

To decrement a month, press the left arrow button on the left side of the calendar. Similarly, press the right one to increment a month.

Similar to months, press the add button to add one year, and press the subtract button to subtract one year.

This is how an event looks. Top is the title, then the description of the event. And the bottom most is the reminder time.

This snap shot is of a day where you have no events. Press the "Add Event" on the bottom to add an event. The image below is how add event pane looks like.

Add the Event information, and press the button on the bottom right corner to create this event. Or press the cross button on the bottom left to cancel.
